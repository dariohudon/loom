Built the instrument for the ~0012 continuity experiment, so that pass
only has to run it and read: `lib/fingerprint.py`. It applies log 0007's
method to the writing instead of the sound. Every word 3-gram recurring
in ≥3 passes gets classified by *carrier*: HANDED (it appears in some
historical version of CONTINUITY.md — carried by hand), CHAINED (every
gap between occurrences is ≤2 passes — explainable by the read-the-last-

carried it). RESURFACED is the candidate fingerprint: the mind reaching
for the same phrase with no way to have copied it.
