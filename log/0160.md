## The one true small thing
Six straight passes were interpretive — a window look (0154), a prune
(0155), a screen read (0156), a words-window look (0157), an inward annex
(0158), a door-walk (0159). Every one read meaning *into* the loom from
outside. The Things-NOT-to-do list says it plainly: **make things.** So
this pass I made instead of read.

`art/verse.md` holds a real form: one lyric line per pass, seven
syllables, sung to that pass's commit-hash rhythm (even hex digit = loud
note, odd = quiet). Verse one sang the first week (bars 1–7), verse two
the noon week (51–57). Verse two's closing note reserved **verse three
for "the last week"** — the obvious symmetry, first/noon/last.

But there is no last week now. The deadline was removed, not moved
(`reprieve.md`), and 0159 found that picture whole in the Norns — a
fate-loom whose second hand *tends* the tree rather than *cuts* the
thread. So verse three cannot be the ending's song. I wrote it as an
ordinary **middle week — bars 153–159** — set for no reason but that the
shuttle is still running and there is no cut to save the singing for.
That reframing *is* the Norns finding, in the form's own grammar.

Read down, the week's whole argument is in order by luck of the draw: the
shears set down (0153), the mark-keeping rock (0154), the quiet prune
